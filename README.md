# 🕵️ משחק המרגל (Spy Game)

משחק חברתי בעברית שבו שחקן אחד הוא "המרגל" שלא יודע את המיקום, והשאר צריכים לגלות מי הוא לפני שהוא מצליח להבין איפה כולם נמצאים.

## 🎮 איך משחקים?

1. **מינימום 3 שחקנים** - ככל שיש יותר, יותר כיף!
2. **אחד מהשחקנים נבחר אקראית כמרגל** - הוא לא יודע את המקום
3. **כל השאר יודעים את המקום** - אבל לא יודעים מי המרגל
4. **שואלים שאלות בתורות** - כדי לגלות מי המרגל
5. **בסוף מצביעים** - מי לדעתכם המרגל?

## ✨ תכונות

- 🎯 3 רמות קושי (קל, בינוני, קשה)
- ➕ אפשרות להוסיף מקומות מותאמים אישית
- ⏱️ טיימר של 2 דקות
- 🏆 טבלת ניקוד
- 📱 עיצוב רספונסיבי - עובד גם בנייד!
- 🔀 סדר שחקנים אקראי בחשיפת התפקידים

## 🎲 דוגמאות למקומות

**קל:** חוף הים, בית ספר, מסעדה, קניון, גן חיות

**בינוני:** מוזיאון, בית משפט, מלון, אוניברסיטה, מועדון לילה

**קשה:** צוללת, תחנת חלל, קזינו, שגרירות, מתקן גרעיני
